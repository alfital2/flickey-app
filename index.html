03 - Features

It just keeps the language right.

macOS has a per-app input setting. It forgets, lags, and resets. FlicKey is what it should have been.

CORE

Per-app language memory

Remembers the right input language for every app and switches the moment you focus it. Terminal stays Latin, chat stays Hebrew.

Per-site memory

Your browser remembers the language for each website, so the right script is armed before you press a single key.

Auto-fix wrong-layout text

Type in the wrong layout and FlicKey fixes it and switches for you, as you type. Double-tap ⇧ converts on demand - any two layouts, any app.

04 - Privacy & security

Private by design.

A tiny local utility, and fully open source - read every line yourself. Not a keylogger: it reads your typing only in the moment to switch or fix a layout.

Stays on your Mac

It only stores tiny preferences - which layout you like per app, website, and chat. No documents, no messages, no keystrokes.

No account, no tracking

No sign-in, no analytics, no telemetry. Nothing is ever sent on its own - there is no server to send to.

One network call

The only connection it makes is a version check to GitHub - and it sends nothing about you. Anything else you might share - a bug report, words it fixed wrongly - exists only as an email that you review and choose to send yourself.

Permissions, explained

Two macOS permissions, each for one clear job. Tap a row to see exactly what it does - and doesn't.
